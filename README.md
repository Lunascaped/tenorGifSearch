
# Tenor Gif Search

A Vencord plugin that (should) allow you to continue using Tenor after the API dies by using the mobile app's API key. [Google has stated the app will still work](https://support.google.com/tenor/answer/10455265#zippy=%2Chow-will-this-affect-users) so this should also let you use it in Discord.

Didn't commit this to main repo because I don't think they'd accept a plugin using an API key like this.
